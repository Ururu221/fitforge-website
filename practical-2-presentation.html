ЧОМУ
FITFORGE?

Більшість фітнес-застосунків пропонують статичні плани. FitForge використовує штучний інтелект для динамічної адаптації тренувань і харчування під кожного користувача в реальному часі.

Цільова аудиторія — люди 18–40 років, які хочуть займатись спортом ефективно, без зайвих витрат часу на планування.

🎯 Проблема

Звичайні застосунки дають однакові плани для всіх. AI-персоналізація досі недоступна масовому користувачу в зручному форматі.

💡 Рішення

FitForge аналізує дані користувача та генерує унікальний план тренувань і харчування, який змінюється разом з прогресом.

📈 Монетизація

Freemium-модель: базовий план безкоштовний, преміум-функції доступні за $9.99–$19.99/місяць.

🔧 Технологія

Лендінг створено на HTML + Tailwind CSS + Vanilla JS. Дизайн розроблено через NanoBanana, код — через dev0 + GitHub.
