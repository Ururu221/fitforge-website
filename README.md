# FitForge 💪

AI-powered fitness app landing page.

## Concept
FitForge is a fitness application that uses AI to create 
personalized workout and nutrition plans.

## Tech Stack
- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript

## Sections
- Hero with animated CTA
- Stats bar
- How It Works (3 steps)
- Features (3 cards)
- Testimonials
- Pricing (Free / Pro / Elite)
- Footer + Mobile sticky nav
